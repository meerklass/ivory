.. :changelog:

History
-------

3.0.0 (2026-01-14)
++++++++++++++++++
* Added support for loading configuration files from file paths (not just module names)
* Configuration files can now be located anywhere on the filesystem
* Supports absolute paths, relative paths, and tilde expansion (~/config.py)
* Maintains full backward compatibility with module-based configuration
* Updated documentation with new usage examples
* Enhanced CLI help text with comprehensive usage information

2.0.0 (2024-XX-XX)
++++++++++++++++++
* Migrated to Python 3.12 and numpy 2.x
* Updated build system to use pyproject.toml
* Improved argument handling for --help and -h flags

0.1.0dev (2014-07-22)
+++++++++++++++++++++
* Consistent time recording
* Syntactic sugar for Workflowmgr calls
* Parallel execution of workflows on IPython clusters

0.1.0 (2014-01-01)
++++++++++++++++++

* First release on PyPI.
